To run the Crop Yield Predictor for Team 2, 'Croply' follow the next steps:

-Download project files to machine (this project runs only locally)
-Open a command line window.
-Navigate to the location path where you downloaded the project files.
-Go to the backend folder to run it.
-Make sure to have the following installed:

pip install flask-cors

pip install joblib

-Run the backend server with:

python app.py

You can follow the given URL that confirms the server is running locally.

To run the frontend, open a code editor like Visual Studio Code:
-Open the project folder.
-Open a new VS Code terminal.
-Navigate to the frontend folder with:

\Crop-Yield-Predictor-\frontend\frontend

-Run:

npm start

A new browser window will pop-up with the application displaying and running locally.

(Make sure to have Node.js installed in your machine)
